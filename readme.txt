== Description ==

ChatNhanh là ứng dụng nhắn tin và kết nối cộng đồng mua bán Muabannhanh.com hàng đầu tại Việt Nam

* Đặc biệt ở ChatNhanh?
* Tốc độ gửi tin nhắn rất nhanh, hệ thống gởi thông báo (notification) tức thời giúp cập nhật ngay cả khi không mở ứng dụng
* Tạo kênh riêng cho các chat thông báo, giúp kết nối cộng đồng nhận tin tức, sự kiện kinh doanh, làm việc.
* Chat với các liên hệ từ danh bạ thật dễ dàng ngay cả khi bạn bè chưa tải ứng dụng.
* An toàn và riêng tư.
* 30s để kích hoạt ChatNhanh và chat ngay với bạn bè, khách hàng, đối tác trong danh bạ
* Tải Ứng dụng  Android:[CHPlay](https://play.google.com/store/apps/details?id=chatnhanh.core)
* Tải Ứng dụng IOS: [AppleStore](https://itunes.apple.com/vn/app/chatnhanh-chat-mua-b%C3%A1n-nhanh/id1061004934?l=vi&mt=8)
* Chat phiên bản website: [ChatNhanh](https://chat.muabannhanh.com/)

== Installation ==

1. Tải thư mục plugin  vào `/wp-content/plugins/` hoặc cài từ WordPress plugins.
2. Kích hoạt plugin từ trang quản lý plugins của  Wordpress
3. Vào ChatNhanh từ menu admin, nhập thông tin admin của live chat.
